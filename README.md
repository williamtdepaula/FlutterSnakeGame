## Configuration Steps

Cloning the repository:
```
$ git clone https://github.com/williamtdepaula/FlutterSnakeGame.git
```

Open the project:
```
$ cd snake_game
```

Now run the app on your connected device (using terminal):
```
$ flutter run
```
